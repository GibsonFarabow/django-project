Chapel Hill Alcoholic Drinks DB

This is a proof-of-concept database that contains information about cocktails and drinks. A full fledged DB like this could help people learn how to make drinks, as well as see which drinks they might want to avoid because of a dietary restriction.

I built this site with Python and Django from the database I made for a class project (and this page doesn't show the many-to-many tables and some others).
